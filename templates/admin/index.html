User Story Application

This application assists developers in incorporating their user stories during project planning. It allows developers to add, mark as completed, and remove user stories as needed. When a developer adds a user story, it is displayed in green color to indicate that the user story is still open. When the user story is marked completed it will crashed and have a red background, this is done by clicking on the user story. The application also provides the ability to remove completed user stories individually or all at once. Additionally, the app includes a feature that allows users to re-add a user story that was previously marked as completed.
